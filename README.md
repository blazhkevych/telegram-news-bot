# UA News Bot

[![Python](https://img.shields.io/badge/Python-3.10%2B-blue)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)
[![Telegram](https://img.shields.io/badge/Channel-UA%20News-blue)](https://t.me/ua_news_channel)

Telegram-бот для автоматичного збору новин з RSS-каналів українських та світових медіа, фільтрації за релевантністю та публікації в каналі [UA News](https://t.me/ua_news_channel).

## Особливості

- **Автоматичний збір новин:** Парсинг RSS-каналів провідних медіа (УНІАН, Укрінформ, BBC Україна, Kyiv Independent тощо).
- **Фільтрація та релевантність:** Видалення дублікатів, перевірка на актуальність, фільтрація за ключовими словами та відповідність тематиці каналу.
- **Дайджести:**
  - **Ранковий дайджест** — ключові події за ніч.
  - **Вечірній дайджест** — підсумки дня.
- **Статистика втрат ЗСУ:** Періодична публікація актуальних даних (оновлюється щодня з відкритих джерел Генштабу ЗСУ).
- **Підтримка кількох джерел:** Можливість легко додавати/видаляти RSS-канали через конфігурацію.
- **Логування:** Детальні логи роботи бота для моніторингу та відлагодження.

## Вимоги

- Python 3.10 або вище
- Залежності з `requirements.txt`

## Встановлення

### Локальне встановлення

1. Клонуйте репозиторій:

   ```bash
   git clone https://github.com/your-username/telegram-news-bot.git
   cd telegram-news-bot
   ```

2. Створіть віртуальне середовище та активуйте його:

   ```bash
   python -m venv venv
   source venv/bin/activate   # Для Linux/Mac
   venv\Scripts\activate      # Для Windows
   ```

3. Встановіть залежності:

   ```bash
   pip install -r requirements.txt
   ```

4. Налаштуйте конфігурацію:

   ```bash
   cp .env.example .env
   ```
   Відредагуйте `.env`, заповнивши необхідні змінні (див. розділ "Конфігурація").

### Docker (опціонально)

```bash
docker build -t ua-news-bot .
docker run -d --env-file .env --name ua-news-bot ua-news-bot
```

## Конфігурація

Створіть файл `.env` у кореневій директорії з наступними змінними:

| Змінна              | Опис                                      | Приклад значення                        | Обов'язкове |
|---------------------|-------------------------------------------|-----------------------------------------|-------------|
| `BOT_TOKEN`         | Токен Telegram-бота (від BotFather)       | `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz` | Так         |
| `CHANNEL_ID`        | ID каналу для публікації новин            | `@ua_news_channel` або `-1001234567890` | Так         |
| `DATABASE_URL`      | URL бази даних (SQLite або PostgreSQL)    | `sqlite:///news.db`                     | Так         |
| `RSS_SOURCES`       | Список RSS-каналів (через кому)           | `https://rss.nytimes.com,https://feeds.bbci.co.uk/news/rss.xml` | Так         |
| `MORNING_DIGEST`    | Час ранкового дайджесту (UTC)             | `06:00`                                 | Ні          |
| `EVENING_DIGEST`    | Час вечірнього дайджесту (UTC)            | `18:00`                                 | Ні          |
| `STATS_INTERVAL`    | Інтервал публікації статистики (хвилини)  | `1440`                                  | Ні          |
| `FETCH_INTERVAL`    | Інтервал збору новин (хвилини)            | `30`                                    | Ні          |
| `FILTER_KEYWORDS`   | Ключові слова для фільтрації (через кому) | `Україна,ЗСУ,війна`                     | Ні          |
| `MAX_AGE_HOURS`     | Максимальний вік новини (години)          | `24`                                    | Ні          |
| `LOG_LEVEL`         | Рівень логування                          | `INFO`                                  | Ні          |

## Використання

### Запуск

```bash
python bot.py
```

Бот запуститься в асинхронному режимі та почне збір новин згідно з налаштованим розкладом.

### Команди бота (в особистих повідомленнях)

| Команда       | Опис                             |
|---------------|----------------------------------|
| `/start`      | Початок роботи з ботом           |
| `/help`       | Довідка з доступних команд       |
| `/status`     | Статус бота та статистика роботи |
| `/sources`    | Список активних RSS-джерел       |
| `/digest`     | Отримати дайджест зараз          |

### Режими роботи

- **Автоматичний режим:** Бот працює за розкладом, публікуючи новини та дайджести у канал.
- **Ручний режим:** Адміністратор може викликати команди для отримання дайджестів або статистики поза розкладом.

## Структура проекту

```
telegram-news-bot/
├── bot.py                  # Головний файл бота (точка входу)
├── config.py               # Конфігурація (завантаження змінних оточення)
├── requirements.txt        # Залежності Python
├── .env.example            # Приклад конфігурації
├── .env                    # Змінні оточення (не включати в репозиторій!)
├── Dockerfile              # Контейнеризація (опціонально)
├── src/
│   ├── __init__.py
│   ├── rss_parser.py       # Парсинг RSS-каналів
│   ├── filters.py          # Фільтрація новин (дублікати, релевантність, ключові слова)
│   ├── digest_generator.py # Генерація дайджестів
│   ├── stats_collector.py  # Збір статистики втрат ЗСУ
│   ├── telegram_handler.py # Взаємодія з Telegram API
│   └── database.py         # Робота з базою даних (SQLAlchemy ORM)
└── tests/                  # Тести
    ├── __init__.py
    ├── conftest.py         # Фікстури для pytest
    ├── test_rss_parser.py
    ├── test_filters.py
    └── test_database.py
```
